# maoxuan 技能升级计划

## 现状分析

### 现有内容
- **SKILL.md**：407行，包含7个核心心智模型、10条决策启发式、表达DNA、人物时间线、价值观与反模式、内在张力、智识谱系、诚实边界
- **references/**：6个研究文档（核心著作提炼、战略思维、表达DNA、外部评价、关键决策、人物时间线）

### 可升级方向
1. **增加实战案例库**：现有应用举例较零散，缺少系统化的现代场景案例
2. **增强决策流程**：从"知道框架"到"知道怎么用"，增加可操作的决策步骤
3. **补充反模式对照**：增加常见错误分析，帮助用户识别自己的思维盲区
4. **增加场景化模板**：针对不同使用场景（创业、管理、个人成长等）提供专用分析模板
5. **优化触发机制**：扩展触发词，让更多相关场景能自动激活
6. **增加对比分析**：与其他思维框架的对比，帮助用户理解适用边界

## 升级方案

### 一、SKILL.md 升级

#### 1. 扩展触发词（第10行附近）
```
新增触发词：
- 矛盾分析、主要矛盾、统一战线、群众路线
- 调查研究、实事求是、持久战、农村包围城市
- 战略分析、竞争策略、组织管理、问题诊断
- 创业决策、团队管理、市场进入、竞争对抗
```

#### 2. 增加"决策工作流"章节（在决策启发式之后）
新增章节：**决策工作流（Decision Workflow）**

提供5个标准决策流程，每个流程包含：
- 适用场景
- 具体步骤（可操作）
- 输出模板
- 常见错误

流程包括：
1. **矛盾分析工作流**：面对复杂问题时的分析流程
2. **竞争策略工作流**：面对竞争对手时的策略制定流程
3. **组织决策工作流**：团队/组织层面的决策流程
4. **个人成长工作流**：个人发展路径的规划流程
5. **冲突处理工作流**：处理分歧和冲突的流程

#### 3. 增加"现代实战案例库"章节（在决策工作流之后）
新增章节：**现代实战案例库（Modern Cases）**

每个案例包含：
- 场景描述
- 矛盾分析
- 策略选择
- 结果与启示

案例类型：
- 创业公司 vs 大厂竞争（拼多多、快手等）
- 团队管理冲突处理
- 产品决策分歧
- 个人职业发展困境
- 市场进入策略

#### 4. 增加"反模式自检清单"（在价值观与反模式章节扩展）
扩展现有反模式内容，增加：
- **思维盲区自检表**：10个常见问题，帮助用户识别自己的思维盲区
- **常见错误模式**：每种反模式的具体表现和纠正方法

#### 5. 增加"场景化分析模板"（在诚实边界之前）
新增章节：**场景化分析模板（Scenario Templates）**

提供5个场景的专用模板：
1. **创业决策模板**
2. **团队管理模板**
3. **市场竞争模板**
4. **个人成长模板**
5. **冲突处理模板**

每个模板包含：
- 核心问题清单
- 分析步骤
- 输出格式

#### 6. 优化表达DNA章节
- 增加更多标志性句式示例
- 增加"避免的表达方式"对照表
- 增加语气强度调节指南
