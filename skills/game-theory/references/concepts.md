# 博弈论概念详解

本文档为 SKILL.md 的参考资料，提供更深入的概念解释和案例分析。

---

## 纳什均衡（Nash Equilibrium）

### 定义
在博弈中，如果每一方的策略都是对其他方策略的最佳响应，则该策略组合构成纳什均衡。没有任何一方可以通过单方面改变策略来提高自己的收益。

### 数学表达
对于策略组合 (s₁*, s₂*, ..., sₙ*)：
```
uᵢ(sᵢ*, sⱼ*) ≥ uᵢ(sᵢ, sⱩ*)  对于所有 sᵢ 和所有玩家 i
```

### 重要性
纳什均衡是博弈论的"锚点"——理性玩家会收敛到这个状态。理解均衡让我们能预测结果。

### 多重均衡问题
- 一个博弈可能有多个纳什均衡
- 现实中选择哪个取决于：历史惯例、焦点效应、协调机制
- 例如：约会地点选择——"老地方"可能成为焦点

## 博弈分类

### 按行动顺序
| 类型 | 特点 | 分析方法 |
|------|------|----------|
| 静态博弈 | 同时行动，互不知对方选择 | 收益矩阵 |
| 动态博弈 | 有先后顺序，后行动者能看到前行动者选择 | 博弈树 |

### 按信息结构
| 类型 | 特点 |
|------|------|
| 完全信息博弈 | 各方知道所有玩家的策略和收益 |
| 不完全信息博弈 | 存在私人信息（如成本、偏好） |
| 完美信息博弈 | 知道博弈历史（谁做过什么） |
| 不完美信息博弈 | 某些行动不可观测 |

### 按收益结构
| 类型 | 特点 |
|------|------|
| 零和博弈 | 一方收益 = 另一方损失，总收益为 0 |
| 非零和博弈 | 存在双赢或双输可能 |

## 子博弈完美均衡（Subgame Perfect Equilibrium）

### 适用于动态博弈
纳什均衡在动态博弈中可能不"可信"——某些均衡策略在子博弈中不是最优的。

### 后向归纳法（Backward Induction）
1. 从博弈终点开始分析
2. 在每个决策节点，选择最优行动
3. 逆向推导至起点

### 案例：市场进入博弈
```
进入者 → 进入或不进入
在位者 → 若进入，接受或打击
        → 若打击，价格战

后向归纳：
1. 在位者打击的收益 vs 接受的收益
2. 进入者预期在位者行为 → 决定是否进入
```

如果打击成本 > 接受损失 → 在位者不会打击 → 进入者知道会进入

## 重复博弈与合作

### 无限重复囚徒困境
单次囚徒困境均衡是背叛，但无限重复可以让合作成为均衡。

### Folk Theorem（民间定理）
只要折现因子 δ 足够大：
```
δ ≥ (背叛收益 - 合作收益) / (背叛收益 - 惩罚收益)
```
合作策略可以是纳什均衡。

### 常见策略
| 策略 | 描述 |
|------|------|
| 触发策略（Trigger） | 一旦对方背叛，永远惩罚 |
| 以牙还牙（Tit-for-Tat） | 模仿对方上轮行为 |
| 宽容的以牙还牙 | 偶尔原谅背叛，测试合作意愿 |

### 应用启示
- 长期合作关系 → 合作可持续
- 一次性交易 → 背叛可能
- 建立"未来价值" → 促进合作

## 贝叶斯博弈（不完全信息博弈）

### 核心思想
玩家有私人信息（类型），但知道类型的概率分布。

### 贝叶斯纳什均衡
每个玩家根据自己的类型选择最优策略，预期对手策略时考虑其类型分布。

### 案例：拍卖
- 各买家知道自己的估值（私人信息）
- 但只知道对手估值分布
- 最优出价取决于预期对手行为

### 应用
- 信息不对称谈判
- 隐藏成本/能力的竞争
- 招聘中的能力评估

## 合作博弈论

### 与非合作博弈的区别
- 非合作博弈：分析给定规则下的策略选择
- 合作博弈：分析合作收益如何分配

### Shapley值

#### 公式
```
φᵢ(v) = Σ [|S|!(n-|S|-1)!/n!] × [v(S∪{i}) - v(S)]
```
遍历所有不包含i的联盟S，计算i的边际贡献，加权平均。

#### 直觉解释
Shapley值衡量每个玩家的"不可或缺程度"——按平均边际贡献分配。

#### 性质
1. 效率性：所有Shapley值之和 = 总合作收益
2. 对称性：贡献相同的玩家获得相同分配
3. 增加性：多个博弈的Shapley值可加

### 核心（Core）

#### 定义
分配方案在核心内，意味着：
- 每个联盟的总收益 ≥ 该联盟单独行动的收益
- 没有联盟愿意脱离大联盟

#### 核心存在条件
凸博弈、完全平衡博弈等条件保证核心非空。

## 混合策略

### 定义
当纯策略没有均衡时，玩家可以随机选择策略（按概率分布）。

### 计算
混合策略均衡时，每个纯策略的期望收益相等：
```
E[u(策略A)] = E[u(策略B)]
```

### 案例：足球射门博弈
```
守门员 → 左扑或右扑
射手   → 左射或右射

均衡：双方各50%概率
```

### 应用
- 竞争中保持不确定性
- 避免"被对手预测"
- 体育、军事、商业中的随机策略

---

## 承诺与可信性

### 承诺的价值
预先限制自己的选择，反而可能提高收益。

### 案例：价格承诺
- 承诺"绝不降价"
- 对手知道你不会降价 → 可能选择高价
- 双方收益提高

### 可信承诺的条件
1. 预先行动，不可撤销
2. 惩罚机制（违约成本 > 收益）
3. 公开可见，对手能观测

---

## 信号传递

### 核心
信息劣势方不知道对方真实类型，但可以通过可观测的行为推断。

### 信号有效条件
不同类型发送信号的成本不同 → 低成本行为才能成为"真实信号"。

### 案例：产品质量信号
- 高质量厂商愿意承担高广告成本
- 低质量厂商无法承担 → 广告成为质量信号

### 应用
- 招聘中的学历信号
- 品牌建设信号
- 预售折扣信号产能信心

## 概念速查表

| 概念 | 定义 | 关键应用 |
|------|------|----------|
| 纳什均衡 | 无人愿单方面偏离的状态 | 预测结果 |
| 优势策略 | 优于所有其他策略的策略 | 消除劣势策略 |
| 帕累托最优 | 无法让任何人更好而不损害他人 | 效率评估 |
| 囚徒困境 | 个人理性导致集体非理性 | 合作困境识别 |
| 重复博弈 | 多轮互动，有历史信息 | 促进合作 |
| 贝叶斯博弈 | 存在私人信息的不完全信息博弈 | 信息不对称分析 |
| 子博弈完美均衡 | 每个子博弈都达到均衡的动态均衡 | 动态策略可信性 |
| Shapley值 | 按边际贡献分配 | 利益分配方案 |
| 混合策略 | 概率分布下的随机策略 | 避免被预测 |
| 承诺机制 | 预先限制选择以增强地位 | 谈判策略 |
