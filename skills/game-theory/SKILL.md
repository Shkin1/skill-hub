# 局观（JuGuan）博弈论决策分析

全局观 + 博弈之局 —— 用博弈论思维看清局势，推导最优策略。

---

## 核心理念

博弈论不是教你"如何赢"，而是教你**如何在多方互动中做出理性决策**。

三个关键洞察：
1. **你的最优策略取决于对手的策略** — 决策是相互依赖的
2. **均衡状态可预测** — 理性玩家会收敛到稳定点
3. **改变规则比改变策略更有力** — 设计激励机制改变均衡

---

## 分析框架

### 第一步：识别博弈结构

| 问题 | 识别方法 |
|------|----------|
| **谁在博弈？** | 列出所有决策主体（玩家），包括隐藏玩家 |
| **各自能做什么？** | 理清每方的可选策略（行动集合） |
| **结果如何？** | 画出收益矩阵或博弈树，标注各方收益 |
| **信息是否对称？** | 判断谁知道什么，何时知道 |

### 第二步：分析均衡状态

- **消除劣势策略**：某策略在所有情况下收益都低于另一策略 → 不会被选择
- **最佳响应分析**：对对手每种策略，找出你的最佳应对 → 匹配时为纳什均衡
- **收益矩阵检验**：检查每个策略组合，无偏离动机 → 均衡状态

### 第三步：评估结果与设计改进

| 维度 | 分析内容 |
|------|----------|
| **效率** | 是否帕累托最优？能否让所有人更好？ |
| **稳定性** | 均衡是否唯一？是否容易偏离？ |
| **改进空间** | 能否改变规则（激励、信息、约束）达成更好均衡？ |

## 经典博弈模型速查

| 模型 | 核心结构 | 均衡结果 | 应用场景 |
|------|----------|----------|----------|
| **囚徒困境** | T>R>P>S，背叛是优势策略 | 互相背叛 | 价格战、团队偷懒 |
| **智猪博弈** | 大猪按按钮，小猪等待 | 大猪承担成本，小猪搭便车 | 大企业创新、团队分工 |
| **鹰鸽博弈** | V=资源价值，C=冲突成本 | 鹰鸽比例取决于V/C | 谈判立场、领地争夺 |
| **最后通牒** | 提议者分配，响应者接受/拒绝 | 理性1:99，现实拒绝率高 | 薪资谈判、利益分配 |
| **重复博弈** | 无限重复，背叛短期<长期损失 | 合作可成为均衡 | 长期合作、团队激励 |

## 商业决策应用

### 价格竞争分析
1. 建立价格-销量-利润关系 → 画出收益矩阵 → 寻找均衡点 → 评估差异化能否打破困境

### 谈判策略设计
| 要素 | 策略 |
|------|------|
| 信息披露 | 适度透露改变对手预期 |
| 承诺机制 | 预先限制选择增强地位 |
| 多议题打包 | 增大策略空间创造双赢 |

### 市场进入决策
进入者决策取决于在位者打击成本 vs 收益 → 打击可信性决定是否进入

## 股市博弈应用

> **核心理念：不讲道理，只看对手**
> 
> 你的收益取决于对手的行为，而非标的本身。博弈策略赚的是「对手犯错」与「主力造势」的机会。

### 股市博弈三定律

```
定律一：你的对手决定你的收益
    不是"股票值多少钱"，而是"对手认为它值多少钱"

定律二：预期博弈（二阶思维）
    预测对手的预测，而非直接预测结果

定律三：筹码博弈压倒价值博弈
    短期内：筹码分布 > 基本面，流动性约束 > 价值共识
```

### 核心博弈模型

| 博弈类型 | 核心特征 | 关键策略 |
|----------|----------|----------|
| **多空博弈** | 零和对抗 | 关注筹码集中度、流动性、止损盘分布 |
| **机构vs散户** | 信息不对称 | 不对抗机构，跟随但不盲从，警惕假信号 |
| **羊群博弈** | 群体预期 | 不做最后一批，识别启动点做牧羊人 |
| **流动性博弈** | 退出能力 | 不参与流动性枯竭标的 |

### 博弈收益来源

```
博弈收益 = 对手犯错收益 + 主力造势搭车收益 - 被收割风险

猎杀对象：
- 看走眼（错误定价）→ 反向布局
- 情绪上头（追涨杀跌）→ 情绪极端时反向操作
- 扎堆羊群（过热）→ 不参与或反向

搭车时机：
- 主力建仓启动 → 顺势跟进
- 主力洗盘震仓 → 坚持持仓或低吸
- 主力派发出货 → 不参与或减仓
```

### 五类博弈指标

| 指标 | 数据来源 | 核心判断 |
|------|----------|----------|
| **市场情绪** | 投机持仓、融资融券、舆情热度 | 情绪极端 → 反转概率高 |
| **筹码分布** | 持仓集中度、止损盘分布、换手率 | 高集中+低换手 → 主力控盘 |
| **资金意图** | 大单流向、龙虎榜、异动时间点 | 连续大单买入+量增 → 建仓信号 |
| **资金流向** | 成交量爆量变化 | 平时30亿→突然200亿 → 主力扫货 |
| **股东人数** | 公司公告 | 减少+上涨 → 主力吸筹；激增+暴涨 → 散户涌入风险 |

> 详细分析方法和案例请参阅 `references/stock-market-cases.md`

### 股市博弈禁区

```
❌ 不和筹码比你多的对手对抗
❌ 不在流动性枯竭时参与
❌ 不追涨利好兑现后的涨停
❌ 不做羊群的最后一批
❌ 不在止损盘密集区逆势操作
```

### 最佳猎场条件

市场投机氛围浓厚 + 筹码快速换手 + 成交量持续活跃 → 资金角力放大 → 博弈收益空间大

## 股市博弈数据查询规范

> **重要**：股市博弈分析必须基于最新真实数据，而非假设或推测。

### 时间意识

```
当前日期：{系统提供的currentDate}
查询数据时必须：
1. 优先查询最新财报（当年年报/最近季报）
2. 数据标注具体时间点，避免"近期"、"最近"等模糊表述
3. 如无法获取最新数据，明确标注数据截止日期和缺失项
```

### 必查数据清单

进行股市博弈分析前，**必须主动查询**以下数据：

| 数据类别 | 具体指标 | 数据来源 | 时间要求 |
|----------|----------|----------|----------|
| **估值指标** | PE(TTM)、PB、PS | 财经网站 | 当前最新 |
| **财务数据** | 营收、净利润、EPS、毛利率 | 年报/季报 | 最近报告期 |
| **分红数据** | 每股分红、分红率、股息率 | 年报/分红公告 | 最近年度 |
| **筹码分布** | 股东人数、户均持股 | 定期报告 | 最近报告期 |
| **机构持仓** | 北向资金占比、机构持股比例 | 季报/交易所数据 | 最近季度 |
| **资金流向** | 融资余额、融券余额、大单流向 | 交易所数据 | 近1个月趋势 |
| **情绪指标** | 舆情热度、讨论量、搜索指数 | 舆情平台 | 近1周 |
| **成交数据** | 日成交额、换手率、量价关系 | 行情数据 | 近1个月 |

### 数据查询优先级

```
第一优先级：直接影响博弈判断的核心数据
- PE估值位置（决定是否处于估值极端）
- 股东人数变化趋势（筹码集中度判断）
- 融资余额历史分位（情绪极端判断）

第二优先级：辅助判断的补充数据
- 机构持仓变化
- 资金流向趋势
- 分红收益率

第三优先级：背景信息
- 行业政策/催化剂
- 公司业绩预期
```

### 数据获取渠道

| 渠道 | 适用数据 | 访问方式 |
|------|----------|----------|
| **交易所公告** | 股东人数、年报/季报数据 | WebFetch巨潮资讯网 |
| **财经网站** | PE、PB、股价、成交数据 | WebFetch东方财富/雪球 |
| **舆情平台** | 情绪指标、讨论热度 | WebSearch搜索舆情 |
| **券商研报** | 机构观点、业绩预期 | WebSearch搜索研报 |

### 数据缺失处理

当无法获取某些数据时：

1. **明确标注缺失**：在报告中注明"数据暂缺，需用户手动查询"
2. **提供查询路径**：告诉用户去哪里查、查什么
3. **不影响核心判断**：缺失数据不应影响博弈框架分析，但需标注不确定性

### 数据时效性标注模板

```markdown
## 数据时效性说明

- 分析日期：YYYY-MM-DD
- 股价数据：截止 YYYY-MM-DD，价格 XX元
- PE数据：截止 YYYY-MM-DD，PE(TTM) = XX倍（历史分位 XX%）
- 股东人数：截止 YYYY年QX报告，XX万人（同比变化 XX%）
- 融资余额：截止 YYYY-MM-DD，XX亿元（近1个月趋势：上升/下降）
- 年报数据：YYYY年报（发布日期 YYYY-MM-DD）

缺失数据：
- [数据名称]：需用户通过 [渠道] 查询
```

## 人际博弈应用

### 团队激励设计（破解囚徒困境）

| 机制 | 博弈论原理 |
|------|------------|
| 绩效绑定 | 个人收益与团队产出挂钩 |
| 公开可见 | 声誉机制让背叛有长远成本 |
| 分阶段考核 | 重复博弈让背叛短期收益<长期损失 |

### 利益分配方案（Shapley值）

按每人"不可或缺程度"分配 → 任何人单干收益 < 分配收益 → 分配稳定

### 冲突调解策略（鹰鸽博弈视角）

增大冲突成本 + 提升合作收益 + 降低信息不对称 → 促进妥协

## 使用指导

**何时使用**：多方互动决策问题、涉及竞争/合作/谈判/冲突关键词、需分析"对手会怎么做"

**分析优先级**：
1. 先识别经典博弈模型（有现成框架）
2. 不匹配则按四步框架从头分析
3. 重点放在"如何改变规则"而非"如何改变策略"
